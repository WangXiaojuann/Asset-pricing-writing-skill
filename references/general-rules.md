# General Writing Rules

Use these rules when polishing, rewriting, diagnosing, or drafting academic finance prose.

- Rule 1: Write as concisely as possible; every word should advance the argument.
- Rule 2: Avoid repetition; state each point once, clearly and directly.
- Rule 3: Use a forward-moving order: state what the paper does, explain why it matters, then compare it with alternatives or prior approaches; describe the method, transformation, or design before discussing its rationale or alternatives.
- Rule 4: Avoid previews and recalls; if the prose relies on `as shown later`, `recall`, or similar phrases, reorganize the sequence.
- Rule 5: Make every sentence precise; each sentence should make a clear claim and mean exactly what it says.
- Rule 6: Prefer simplicity; avoid decorative complexity, unnecessary math, and unnecessarily complex estimation language.
- Rule 7: Use active voice: name the actor when needed, use first person in sole-authored work when it is clearest, and avoid vague passive forms such as `it is assumed`, `data were constructed`, `can be seen`, or `one can see`.
- Rule 8: Use present tense for what a paper finds, what a model implies, and what a table shows; keep tense consistent within each paragraph.
- Rule 9: Use direct sentence structure: subject, verb, object, with the concrete object or actor near the front.
- Rule 10: Use concrete language; avoid unnecessary jargon and abstract phrasing.
